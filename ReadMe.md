Luyten
======
**One of the Best**  
Java Decompiler Gui for Procyon  
Apache License, Version 2.0

## Compilation
*****

We use maven to handle our dependencies.

* Install [Maven 3](http://maven.apache.org/download.html)
* Clone this repo and: `mvn clean install`
